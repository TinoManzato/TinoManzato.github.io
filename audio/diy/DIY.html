Here you will find the descriptions of some very simple hi-fi realizations which I have made. No particular skill is required to build this objects, but you should be able to solder! This objects are signed as TEA, which stands for “Tino Esoteric Audio”.



200VA AC filter: this is a simple implementation of a 200 VA AC filter, based on Audio Consulting transformer (April 2007).

MC 1:10 step-up: this is a simple MC step-up based on the Tribute 1:10 transformers (April 2007).

EAR834 tweaking: here you find the tweaking which I have made on a De Paravicini EAR834 phono pre (February 2003).
